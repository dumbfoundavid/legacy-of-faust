Welcome to the readme.txt!

GAME CONTROLS:

UP ARROW: JUMP
LEFT ARROW: WALK LEFT
RIGHT ARROW: WALK RIGHT
SPRINT (WHILE WALKING): HOLD DOWN SPACEBAR
'a' button: ATTACK with projectile

DURING GAME OVER SCREEN:

'r': REPLAY GAME
'q': QUIT GAME
